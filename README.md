# Network Topology Builder

A simple interactive web application that allows users to build and visualize basic network topologies directly in the browser.

This project was built using **HTML, CSS, and JavaScript** to demonstrate front-end interaction and basic networking logic.

## Live Demo (for some reason the github one didnt work so I went with netlify till I check this out later)

https://brainther-cpu.netlify.app

## Features

- Drag and drop network devices onto the workspace
- Connect devices together
- Valid connections appear **green**
- Invalid connections appear **red**
- Rename devices with double-click
- Delete devices and automatically remove their connections
- Move devices freely on the canvas

## Supported Devices

- Router
- Switch
- Server
- PC
- Firewall

## Technologies Used

- HTML
- CSS
- JavaScript
- SVG for drawing network connections

## Purpose of the Project

I originally created this project in **2024** while preparing for a **web development internship at an IT company**. At the time, I wanted to build something practical that combined my interest in networking with front-end development.

Recently, I revisited and improved the project to share what I have learned since then and to continue developing it instead of leaving it unfinished. It reflects my ongoing learning process in web development, UI interaction, DOM manipulation, and basic networking concepts.

## How to Use

1. Drag a device from the left panel to the workspace.
2. Click **Connect** to start creating connections.
3. Select two devices to connect them.
4. Double-click a device to rename it.
5. Right-click a device to delete it along with its connections.
